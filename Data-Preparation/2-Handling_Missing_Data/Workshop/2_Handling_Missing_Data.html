Handling Missing Data¶

  • Il s'agit d'un problme rpandu dans le monde de l'analyse de donnes.

  • Des donnes manquantes peuvent survenir dans l'ensemble de donnes pour plusieurs raisons :
    • les donnes du champ spcifique n'ont pas t ajoutes par l'utilisateur/l'application de collecte de donnes, les donnes ont t perdues lors du transfert manuel, une erreur de programmation, etc.
    • Il est parfois essentiel d'en comprendre la cause. car cela influencera la faon dont vous traitez ces donnes.
  • Pour les donnes numriques, pandas utilise une valeur virgule flottante NaN (Not a Number) pour reprsenter les donnes manquantes.
  • Il s'agit d'une valeur unique dfinie dans la bibliothque Numpy , nous devrons donc galement l'importer.
  • NaN est le marqueur de valeur manquante par dfaut pour des raisons de rapidit de calcul et de commodit.

Remplir les données manquantes¶

  • Pour remplacer ou plutt "remplir" les donnes nulles, vous pouvez utiliser la fillna()fonction.
  • Par exemple, essayons d'utiliser le mme ensemble de donnes que ci-dessus et essayons de remplir les valeurs NaN avec 0.
