Comment l'améliorer ?¶

La rgularisation est utilise pour viter le overfitting (over-training).

Pnalit : La norme utilise dans la pnalisation.

  • L1 - suppose que seules quelques caractristiques sont importantes.
  • L2 - ne suppose pas que seules quelques caractristiques sont importantes - utilis par dfaut dans la rgression logistique de scikit-learn.

'C' : Paramtre permettant de contrler la force de la rgularisation. Par dfaut, C est gal 1.

  • C infrieur => log_reg s'ajuste la majorit des points de donnes.
  • C plus lev => classification correcte de chaque point de donnes.

Pouvons-nous travailler avec les paramètres pour l'améliorer encore ?¶

Instanciez la rgression logistique avec le paramtre C.

  1. Le paramtre C permet de contrler la force de la rgularisation. Par dfaut, C est gal 1.
  2. Paramtre Max_iter Nombre maximum d'itrations ncessaires pour que les solveurs convergent.
  3. Solveur {'newton-cg', 'lbfgs', 'liblinear', 'sag', 'saga'}, default='lbfgs : Algorithme utiliser dans le problme d'optimisation.
  • Pour les petits ensembles de donnes, 'liblinear' est un bon choix, tandis que 'sag' et 'saga' sont plus rapides pour les grands ensembles.
  • Pour les problmes multiclasses, seuls 'newton-cg', 'sag', 'saga' et 'lbfgs' grent les pertes multinomiales ;

Les modles linaires font des prdictions bases sur une fonction linaire des caractristiques d'entre : y = (w * x) + b.

  • W est le poids (ou le coefficient accessible via .coef_).
  • B est le offset (ou l'interception accessible via .intercept_) de l'axe des y. Les deux sont des paramtres appris.
  • Y est la prdiction (ou la dcision).

Exemple pour un ensemble de donnes avec une seule caractristique d'entre : y = (w[0] * x[0]) + b.

La rgression logistique est un cas spcifique de modle linaire et son quation se prsente comme suit : y = (w[0] * x[0]) + (w[1] * x[1]) + ... + (w[p] * x[p]) + b.

Les modles linaires fonctionnent bien avec les grands ensembles de donnes, ils sont rapides former et rapides prdire. Cependant, il existe d'autres modes qui se gnralisent mieux dans les espaces de faible dimension.
