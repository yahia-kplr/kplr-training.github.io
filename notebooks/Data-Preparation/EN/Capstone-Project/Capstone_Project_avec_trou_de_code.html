  • Pour faire simple, vous appliquerez ces mthodes d'encodage uniquement sur la carriercolonne.

  • L'ide est que vous avez la libert de choisir les numros que vous souhaitez attribuer aux catgories en fonction du cas d'utilisation commerciale.

  • Vous allez maintenant crer un dictionnaire qui contient des numros de mappage pour chaque catgorie dans la carriercolonne :

Encodage One-Hot¶

  • La stratgie de base consiste convertir chaque valeur de catgorie dans une nouvelle colonne et attribuer une valeur 1ou 0(Vrai/Faux) la colonne.

  • Cela a l'avantage de ne pas pondrer une valeur de manire incorrecte.

  • Il existe de nombreuses bibliothques qui prennent en charge l'encodage chaud, mais la plus simple utilise la mthode pandas' .get_dummies().

  • Cette fonction est nomme ainsi car elle cre des variables factices/indicatrices (1 ou 0).

  • Il y a principalement trois arguments importants ici, le premier est le DataFrame sur lequel vous voulez encoder, le deuxime tant l' columnsargument qui vous permet de spcifier les colonnes sur lesquelles vous voulez faire l'encodage, et le troisime, l' prefixargument qui vous permet de spcifier le prfixe pour le nouvelles colonnes qui seront cres aprs l'encodage.

Encodage binaire¶

  • Cette technique n'est pas aussi intuitive que les prcdentes. Dans cette technique, les catgories sont d'abord codes en ordinal, puis ces entiers sont convertis en code binaire, puis les chiffres de cette chane binaire sont diviss en colonnes spares.

  • Cela encode les donnes dans moins de dimensions que one-hot.

  • Vous pouvez effectuer un encodage binaire de plusieurs manires, mais la plus simple consiste utiliser la category_encodersbibliothque.

  • Vous devez d'abord importer la category_encodersbibliothque aprs l'avoir installe.

  • Appelez la BinaryEncoderfonction en spcifiant les colonnes que vous souhaitez encoder, puis appelez la .fit_transform()mthode dessus avec le DataFrame comme argument.
